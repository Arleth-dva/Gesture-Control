# README – Mini Artículo *GestureControl*

Este directorio contiene el mini artículo **“GestureControl”**, un documento en formato PDF que resume el funcionamiento, metodología y resultados del sistema de reconocimiento de gestos en tiempo real, utilizando MediaPipe Hands y heurísticas geométricas.

## Contenido del documento
El mini artículo incluye:
- Objetivo del proyecto
- Tecnologías utilizadas
- Arquitectura y pipeline de detección
- Reglas geométricas para la clasificación de gestos
- Representación educativa dentro de Minecraft
- Resultados, discusión y trabajo futuro

## Propósito
Este documento sirve como referencia técnica resumida y como parte de la entrega académica del Proyecto NEXO.
